# Images Folder

Place your app logo and other images here.

## Logo File
Please save your Agasthi Mobile logo image as:
- `logo.png` (recommended)
- or `logo.jpg`

The logo should be placed in this directory: `assets/images/logo.png`

## Supported Formats
- PNG (recommended for logos with transparency)
- JPG/JPEG
- SVG (if using flutter_svg package)
